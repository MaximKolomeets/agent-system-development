# TARGET_REPOSITORY_ADOPTION_GUIDE (redirect)

Канон adoption в существующий target repository перемещён в `docs/agent-system/ADOPTION_GUIDE.md` (PR-C4 консолидации методологии, решение архитектора Р3).

Пошаговый перенос методологии в уже существующий repository теперь живёт там — раздел «Пошаговый existing-repo adoption». Режимы adoption, `methodology_reference`, transfer manifest, source snapshot drift, branch-flow канон, minimal first PR — в соответствующих разделах `ADOPTION_GUIDE.md`. Feedback и его sanitization — `docs/agent-system/METHODOLOGY_FEEDBACK_LOOP.md`.

Этот файл сохранён как redirect-заглушка для исторических ссылок (append-only журнал, DECISION_LOG, agent reports) и внешних bookmark. Новые ссылки делать на канон.
