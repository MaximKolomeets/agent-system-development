# TIME_LEDGER_TEMPLATE

## Назначение

Шаблон сводного ledger для недельного, месячного, release или project rollup.
Используется в target repository или в methodology repository, если архитектор
явно включает metrics/accounting в scope задачи.

## Notes

- Legacy entries without accounting fields are marked `legacy/advisory`.
- Private billing data, credentials, account ids and client rates are not stored
  in public methodology repository.
- If token/cost source is unavailable, use `not_available` and explain the
  confidence level.
