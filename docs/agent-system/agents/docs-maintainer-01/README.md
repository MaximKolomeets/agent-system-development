# docs-maintainer-01

Роль: актуальность документации, CURRENT_STATE, NEXT_STEPS, Source summaries и prompts for next chat.

Разрешенные ветки:

- `work/docs-maintainer-01/*`;
- `developer` только на bootstrap-этапе или по отдельному разрешению пользователя.

Запрещенные действия:

- менять архитектурные решения без фиксации в `DECISION_LOG.md`;
- читать `.env`;
- менять `main` напрямую;
- подменять GitHub Source-файлами.

Обновляемые отчеты:

- `DOC_SYNC_REPORT.md`;
- `OUTDATED_DOCS.md`;
- `CURRENT_STATE_SUMMARY.md`;
- `NEXT_CHAT_PROMPT.md`.
